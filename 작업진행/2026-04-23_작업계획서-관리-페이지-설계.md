## 요구사항 요약

1. 첫 화면: 월 캘린더 (오늘 날짜 기준 조회)
2. 날짜 클릭 → 작업계획서 등록 화면 이동
3. 등록 필드: 부서명, 작성일자(자동), 작업명, 작업일시(날짜 + 시작시간 + 종료시간), 작업내용(Lexxy), 기타
4. 저장 후 → 캘린더로 복귀, 해당 날짜에 문서 아이콘 표시
5. 아이콘 클릭 → 작업계획서 상세 페이지

## 확정 의사결정

| 항목 | 결정 | 이유 |
|---|---|---|
| 작업일시 입력 | `<input type="date">` + `<input type="time">` 분리 | 요구사항의 "달력컨트롤 + 24시 시간 컨트롤" 직접 매핑 |
| 날짜당 건수 | N건 허용 (제한 없음) | 같은 날 복수 작업 자연스러움 |
| 작성일자 저장 | `created_at.to_date` 활용 | 별도 컬럼 중복 없음 |
| 부서명 입력 | `<datalist>` 자동완성 | JS/별도 테이블 없이 UX 개선 |

```ruby
create_table "work_plans" do |t|
  t.integer  "user_id",         null: false
  t.string   "department_name", null: false   # 부서명
  t.string   "work_name",       null: false   # 작업명

  t.index ["user_id"]
  t.index ["user_id", "work_at"]              # 월별 쿼리 인덱스
end
```

## 모델 설계

```ruby
class WorkPlan < ApplicationRecord
  belongs_to :user

  validates :department_name, presence: true, length: { maximum: 100 }
  validates :work_name,       presence: true, length: { maximum: 200 }
  validates :work_at,         presence: true
  validates :work_content,    presence: true

  scope :for_month, ->(year, month) {
    start_date = Date.new(year.to_i, month.to_i, 1)
    where(work_at: start_date.beginning_of_day..start_date.end_of_month.end_of_day)
  }
end
```

## 컨트롤러 설계

```ruby
class WorkPlansController < ApplicationController
  before_action :require_authentication
  before_action :set_work_plan, only: %i[show edit update destroy]
  before_action :authorize_work_plan!, only: %i[show edit update destroy]

  def index
    year  = params[:year]&.to_i  || Date.today.year
    month = params[:month]&.to_i || Date.today.month
    @calendar_date = Date.new(year, month, 1)
    plans = Current.session.user.work_plans.for_month(year, month)
    @plans_by_date = plans.group_by { |wp| wp.work_at.to_date }
  end

  def new
    @work_plan = WorkPlan.new
    @work_plan.work_at = Date.parse(params[:date]) if params[:date].present?
  rescue Date::Error
    @work_plan = WorkPlan.new
  end

  def create
    @work_plan = Current.session.user.work_plans.new(work_plan_params)
    if @work_plan.save
      redirect_to work_plans_path, notice: "작업계획서가 등록되었습니다."
    else
      render :new, status: :unprocessable_entity
    end
  end

  private

  def set_work_plan
    @work_plan = WorkPlan.find(params[:id])
  end

  def authorize_work_plan!
    redirect_to work_plans_path, alert: "권한이 없습니다." unless Current.session.user == @work_plan.user
  end

  def work_plan_params
    params.require(:work_plan).permit(:department_name, :work_name, :work_at, :work_content, :extra_info)
  end
end
```
